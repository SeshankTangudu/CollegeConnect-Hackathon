## Overview

CollegeConnect is a role-based college event management prototype developed during my 1st year B.Tech hackathon.

The goal of the project was to design a centralized platform that simplifies event creation, team formation, venue booking, and event browsing for students and administrators.

This version represents the initial frontend prototype built during the hackathon.

---

## Key Features

- Role-based login system (Student / Admin / Super Admin)
- Event creation interface
- Event browsing page
- Team formation module
- Venue booking simulation
- File upload simulation
- Session-based navigation using `sessionStorage`

---

## Tech Stack

- HTML
- CSS
- JavaScript
- Git & GitHub for version control
- GitHub Pages for deployment

---

## Architecture Notes

- Pure frontend prototype
- No backend integration
- No database connectivity
- Role simulation handled using browser session storage

---

## Limitations

- Data is not persistent
- No real authentication system
- Prototype-level functionality only

---


## Project Context

This project represents my early exploration into web development and system design during my first year. Future versions will focus on scalability, backend integration, and full-stack implementation.
